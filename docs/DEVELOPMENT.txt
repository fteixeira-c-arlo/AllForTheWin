ArloShell — developer notes
======================================

Graphical app (PySide6) for Arlo camera devices. Rich / questionary are still used
behind the scenes for shared command menus and prompts in the GUI.

Requirements
------------
- Python 3.10+ for local runs
- ADB: Android SDK platform-tools on PATH (optional)
- SSH: Paramiko only

Quick start (from repo root)
----------------------------
  pip install -r requirements.txt
  python main_gui.py

Device CLI commands load by product line: core/command_profiles.json maps profile ids
to JSON files (E3 Wired → e3_wired_commands.json). core/camera_models.py assigns
command_profile per device row. Unknown detected models get profile "none" (no device
catalog — only shared tools). See core/HOW_TO_ADD_DEVICE_COMMANDS.txt to add lines.

Project layout
--------------
  main_gui.py — application entry point
  core/camera_models.py — camera model groups and profiles
  transports/ — ADB, SSH, UART, connection_config
  core/ — definitions and parser
  interface/ — menus, prompts, GUI
  utils/ — validators, logger
